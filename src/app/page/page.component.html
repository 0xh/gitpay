Gitpay is the platform for descentralized companies and nomad developers.

How it works

  • We automate payment on code changes using git
  • We link developers and content writers, technical documentation, translations and services on demmand
  • Request a feature, bugfix and a developer that can make your job done using the Git Workflow

What is Gitworkflow and what is the relation with all these stuff?

  • With Gitworkflow, you can make changes in the code, infrastructure and ask for a integration with reviews.
  • So Gitpay get into the scene, and when the code is approved, you receive a automatic payment.

  • Github
  • Trello
  • Stripe
