GITPAY

a platform for decentralized companies and nomad developers

ObjectiveOffer a marketplace for developers, digital products and platforms that use GIT for on demand project needs.

How it worksGitpay is meant for products whose code reside in a GIT repo. The client creates an issue for a given price and the developer creates a pull request; once the client approves it, the developer’s code is integrated in the client’s code base, and a payment is automatically sent to the developer.

PurposeClients want to create on demand work, or even a entire project developed iteratively. Developers want to do their jobs and contribute with projects with a good infrastructure. The developer don’t need to work for one centralized company. The existent freelancers platforms still has the downside to work with agreement about unclear requirements. Git workflow solve this.
